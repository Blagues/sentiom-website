### Font Family
```
Primary: Inter
Fallback: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
```

### Font Weights
- **Regular:** 400 (body text)
- **Medium:** 500 (navigation, subtle emphasis)
- **Semi-Bold:** 600 (subheadings, emphasis)
- **Bold:** 700 (headings, buttons, strong emphasis)

### Font Sizes

#### Hero Section
- **H1:** 72px / Bold (700) / Line-height: 1.1
- **Highlight word:** Same size, color: #343FD5

#### Buttons
- **Primary/Secondary:** 18px / Bold (700)
- **Navigation CTA:** 16px / Bold (700)

#### Navigation
- **Menu items:** 16px / Medium (500)
- **Logo:** 20px / Bold (700)

#### Content Sections
- **H2:** 36px / Bold (700) / Line-height: 1.2
- **Body:** 18px / Regular (400) / Line-height: 1.6
- **Small text:** 16px / Regular (400)

## üéØ Usage Guidelines

### Primary Blue (#343FD5)
**DO:**
- Use for all primary CTAs and important actions
- Use for the word "zonder" in hero heading
- Use for hover states on navigation
- Use for links and interactive elements

**DON'T:**
- Don't use for large background areas
- Don't use for body text (readability)

### White (#FFFFFF)
**DO:**
- Main background color
- Text on blue buttons/backgrounds
- Cards and content containers

### Typography
**DO:**
- Use Inter for all text
- Keep heading hierarchy consistent
- Use bold (700) for all buttons
- Use medium (500) for navigation

**DON'T:**
- Don't mix different font families
- Don't use font weights below 400
